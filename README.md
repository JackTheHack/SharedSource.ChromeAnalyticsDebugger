# SharedSource.ChromeAnalyticsDebugger
Chrome extension that adds tab in developer tools to view the Sitecore analytics information (profiles, geolocation data, etc.)

  <img src="https://raw.githubusercontent.com/JackTheHack/SharedSource.ChromeAnalyticsDebugger/master/SharedSource.Extension.ChromeAnalyticsDebugger/src/img/screenshot.JPG" width="100%"/>

# Installation

- Install the Chrome extension
https://chrome.google.com/webstore/detail/sitecore-analytics-develo/oahfemdnmipdocmlcmhkenejhmklgefp?hl=en-GB
- Install the Sitecore package in the target website
[Sitecore Package - SharedSource.AnalyticsChromeDebugger.zip](https://github.com/JackTheHack/SharedSource.ChromeAnalyticsDebugger/files/923997/SharedSource.AnalyticsChromeDebugger.zip)
- Configure the secret key in the config file
